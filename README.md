# Google Ads Brain Full App

## Render setup

Root Directory: leave blank

Build Command:
```bash
pip install -r requirements.txt
```

Start Command:
```bash
gunicorn app:app
```

Environment variable:
```bash
OPENAI_API_KEY=your_key_here
```

## Features

- Frontend UI
- AI ad generation
- Buyer psychology map
- Persuasion assets
- Feedback logging
- Learning loop analysis
- Generate improved ads from feedback

## Main URL

Open your Render URL and the frontend should load.
